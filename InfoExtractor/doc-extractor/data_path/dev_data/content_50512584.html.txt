前5月全国民航完成旅客运输量2.68亿人次 	B-TITLE
2019-06-14 06:45:51 来源： 0 条评论 	B-INFO
新华社北京6月13日电（齐中熙、王亦玮）记者13日从中国民用航空局新闻发布会上获悉，1至5月，全国民航共完成旅客运输量2.68亿人次，同比增长8.5%。从航线结构看，国内、国际航线分别完成2.38亿人次和3020.7万人次，同比分别增长7.6%、16.5%。 	B-CONTENT
民航局发展计划司副司长董法鑫表示，5月份，航空运输市场总体增速有所回升。全行业共完成运输总周转量107.8亿吨公里，同比增长7.9%，增速较上月提高3.3个百分点。其中，国内、国际航线分别完成68.5亿吨公里、39.3亿吨公里，同比分别增长7.8%、8.1%。 	I-CONTENT
航空客运市场有所提升，国际航线旅客增速较快。董法鑫表示，5月，全行业共完成旅客运输量5450.8万人次，同比增长8.7%，增速较上月提升4.1个百分点。其中，国内、国际航线分别完成4847.7万人次、603.1万人次，同比分别增长7.6%、18.7%，国际航线客运增长态势良好。（完） 	I-CONTENT
